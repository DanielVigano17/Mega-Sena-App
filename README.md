# 🎲 Gerador de Números - Mega Sena

Aplicativo Android que realiza o sorteio de 6 números aleatórios entre 1 e 60, 
sem repetição e exibidos em ordem crescente.

---

## 📱 Funcionalidades

- ✅ Sorteia 6 números entre 1 e 60
- ✅ Não repete números
- ✅ Exibe os números em ordem crescente
- ✅ Botão para limpar os números sorteados
- ✅ Interface simples e intuitiva

---

## 🛠️ Tecnologias Utilizadas

- Java
- Android SDK
- Android Gradle Plugin 8.13.1
- Gradle 8.x
